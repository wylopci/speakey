# 🎙️ Speakey

**押して話す、離せばもう入力完了。**

Mac 向け AI 音声入力——あらゆるアプリの入力欄で、キーボードの代わりに声を使う。
「えっと」「あの」などのフィラーを自動で除去し、句読点を補い、リアルタイム翻訳もこなす。

## ✨ 機能

- **🎙 押して話す**:右 Option キーを押しながら話し、離してから1〜2秒でカーソル位置に自動でテキストが入力される——LINE、メール、ブラウザ、メモ、どこでも
- **🧹 AI による整文**:単なる文字起こしではなく、あなたが「本当に伝えたいこと」に整える——フィラーや言い淀み、繰り返しを自動でクリーンにし、句読点も自動補完
- **🌏 多言語対応**:中国語(台湾)/英語/日本語/韓国語+自動判定——話した言語でそのまま入力
- **🔁 リアルタイム翻訳**:中国語で話して英語で出力——英文メールや海外顧客への返信に最適
- **🎯 場面に応じたトーン**:メールに貼れば自動でフォーマルに、チャットに貼れば自動でカジュアルに——場面に合わせて言葉遣いを調整
- **🧠 使うほど精度が上がる**:あなたの端末内であなたの言い回しを学習し、よく使う単語を自動提案;「聞き間違い ⇒ 正しい表記」の修正ルールで頑固な誤変換も根本解決
- **🕶 プライバシー設計**:アカウント不要、追跡なし、自社サーバーなし;使用履歴は端末内にのみ保存され、いつでもオフ・削除が可能

## 📥 ダウンロードとインストール

1. **[Releases](https://github.com/wylopci/speakey/releases/latest)** から該当するバージョンをダウンロード:

   | お使いの Mac(→ この Mac について) | ダウンロード |
   |---|---|
   | チップ:Apple M1 / M2 / M3 / M4 | `Speakey-AppleSilicon.dmg` |
   | プロセッサ:Intel Core i5 / i7 / i9 | `Speakey-Intel.dmg` |

2. **`.dmg` をダブルクリック**し、Speakey のアイコンを「アプリケーション」フォルダにドラッグ
3. 「アプリケーション」を開き**ダブルクリックで起動**(Apple の公証済みのため警告は表示されません)
4. 画面の指示に従う:無料の Groq API キーを貼り付け(取得方法はアプリ内で案内)→ 権限を許可 → 使用開始

詳しい手順は DMG 内のインタラクティブな《インストールガイド》をご覧ください(繁体字中国語/English/日本語/한국어に対応、システム言語に応じて自動切り替え)。

## 🔑 料金

**Speakey は完全無料です。** 音声認識はご自身で取得した [Groq](https://console.groq.com) の無料枠を使用し、日常利用には十分な量です——あなたの API キー、あなたのデータ、あなたの管理下にあります。

## 🛡 プライバシーとセキュリティ

**クラウドを経由する部分(機能上必須)**
- 話した内容と認識されたテキストは、暗号化された状態で Groq に送信され、音声認識と AI による整文/翻訳に使用されます
- デフォルトのモデルは OpenAI がオープンに公開した `whisper-large-v3`(音声認識)と `gpt-oss-120b`
  (文章整形)で、いずれも Groq(米国企業)のサーバー上で実行されます——**デフォルトでは中国企業が開発した AI モデルは一切使用しません**
- メニューから他のモデルに自分で切り替えることも可能です(各選択肢には開発元と国名を明記しており、中国企業のモデルも選択できます);切り替え後は選択したモデルが適用されます
- Groq は公式に、データを保持せず学習にも使用しないと表明しています

**100% 端末内にとどまり、絶対にアップロードされない部分**
- あなたの Groq API キー(ご自身のユーザーディレクトリ内に保存)
- 使用履歴、よく使う単語の学習結果、独自用語辞書、修正対照表
- 実行ログ

**開発者側について**
- Speakey は自社サーバーを持たず、コード内にアップロード処理は一切存在しません
- データは「あなたの端末」と「あなた自身が取得した API キーで接続する Groq」の間でのみやり取りされ、開発者はいかなる使用内容も閲覧・収集できません
- キーボード監視は「ショートカットが押されたかどうか」のみを判定し、キー入力内容は一切記録されません;パスワード入力欄は macOS の Secure Input によって保護されており、音声入力は原理上誤って入り込むことができません

## 💻 動作環境

- macOS 12(Monterey)以降;「ログイン時に自動起動」には macOS 13 以降が必要
- Apple Silicon または Intel チップ(該当するバージョンをダウンロードしてください)
- インターネット接続が必要(音声認識はクラウドで実行されます)

- 「ローカル音声認識モデルを削除」オプションを追加(約 550MB を解放)
