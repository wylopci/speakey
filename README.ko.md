# 🎙️ Speakey

**누르고 말하면, 손을 떼는 순간 이미 입력되어 있습니다.**

Mac용 AI 음성 입력——어떤 앱의 어떤 입력창에서든 키보드 대신 목소리를 사용하세요.
"음", "어" 같은 군말을 자동으로 제거하고 문장 부호를 채워주며, 실시간 번역까지 가능합니다.

## ✨ 기능

- **🎙 누르고 말하기**:오른쪽 Option 키를 누른 채 말하고, 손을 떼면 1~2초 후 커서 위치에 텍스트가 자동으로 나타납니다——LINE, 이메일, 브라우저, 메모 어디서든
- **🧹 AI 스마트 다듬기**:단순한 받아쓰기가 아니라 당신이 "전달하고 싶은 말"로 정리해줍니다——군말, 말더듬, 반복을 자동으로 정리하고 문장 부호도 자동으로 채워줍니다
- **🌏 다국어 지원**:중국어(대만)/영어/일본어/한국어 + 자동 감지——말하는 언어 그대로 입력됩니다
- **🔁 실시간 번역**:중국어로 말하면 영어로 출력——영문 이메일 작성이나 해외 고객 응대에 최적
- **🎯 상황별 어조**:이메일에 붙여넣으면 자동으로 격식 있는 문체, 메신저에 붙여넣으면 자동으로 구어체——상황에 맞춰 말투를 조정합니다
- **🧠 쓸수록 정확해짐**:당신의 컴퓨터 안에서 자주 쓰는 표현을 학습해 자동으로 제안;"잘못 인식된 단어 ⇒ 올바른 단어" 교정 규칙으로 고질적인 오타를 근본적으로 해결
- **🕶 프라이버시 설계**:계정 불필요, 추적 없음, 자체 서버 없음;사용 기록은 오직 당신의 컴퓨터에만 저장되며 언제든 끄거나 삭제할 수 있습니다

## 📥 다운로드 및 설치

1. **[Releases](https://github.com/wylopci/speakey/releases/latest)** 에서 해당 버전을 다운로드하세요:

   | 사용 중인 Mac(→ 이 Mac에 관하여) | 다운로드 |
   |---|---|
   | 칩:Apple M1 / M2 / M3 / M4 | `Speakey-AppleSilicon.dmg` |
   | 프로세서:Intel Core i5 / i7 / i9 | `Speakey-Intel.dmg` |

2. **`.dmg` 파일을 더블클릭**한 뒤 Speakey 아이콘을 "응용 프로그램" 폴더로 드래그하세요
3. "응용 프로그램"을 열고 **더블클릭하여 실행**하세요(Apple 공증을 완료했으므로 경고가 표시되지 않습니다)
4. 화면의 안내를 따르세요:무료 Groq API 키를 붙여넣고(앱이 발급 과정을 안내해 드립니다) → 권한 허용 → 사용 시작

자세한 절차는 DMG 안의 인터랙티브 《설치 안내서》를 참고하세요(繁體中文/English/日本語/한국어 지원, 시스템 언어에 따라 자동 전환).

## 🔑 비용

**Speakey는 완전히 무료입니다.** 음성 인식은 본인이 직접 발급받은 [Groq](https://console.groq.com) 무료 사용량을 사용하며, 일상적인 사용에 충분합니다——당신의 키, 당신의 데이터, 당신이 직접 관리합니다.

## 🛡 개인정보 보호 및 보안

**클라우드를 거치는 부분(기능 작동에 필수)**
- 말한 내용과 인식된 텍스트는 암호화되어 Groq로 전송되며, 음성 인식과 AI 다듬기/번역에 사용됩니다
- 기본 모델은 OpenAI가 오픈소스로 공개한 `whisper-large-v3`(음성 인식)와 `gpt-oss-120b`
  (텍스트 다듬기)이며, 모두 Groq(미국 기업)의 서버에서 실행됩니다——**기본 설정에서는 중국 기업이 개발한 AI 모델을 전혀 사용하지 않습니다**
- 메뉴에서 다른 모델로 직접 전환할 수 있습니다(각 선택지에 개발사와 국가를 명시하며, 중국 기업 모델도 선택 가능);전환 후에는 선택한 모델이 적용됩니다
- Groq는 공식적으로 데이터를 보관하지 않으며 학습에도 사용하지 않는다고 명시하고 있습니다

**100% 당신의 컴퓨터에만 남고 절대 업로드되지 않는 부분**
- Groq API 키(본인의 사용자 디렉터리에 저장)
- 사용 기록, 자주 쓰는 표현 학습 데이터, 사용자 정의 용어집, 교정 대조표
- 실행 로그

**개발자 측면**
- Speakey는 자체 서버가 없으며, 코드 어디에도 업로드 로직이 존재하지 않습니다
- 데이터는 오직 "당신의 컴퓨터"와 "당신이 직접 발급받은 키로 연결된 Groq" 사이에서만 오가며, 개발자는 어떠한 사용 내용도 볼 수 없고 수집할 수도 없습니다
- 키보드 감지는 "단축키가 눌렸는지 여부"만 판단하며 키 입력 내용을 전혀 기록하지 않습니다;비밀번호 입력란은 macOS Secure Input으로 보호되어 음성 입력이 원천적으로 잘못 입력될 수 없습니다

## 💻 시스템 요구 사항

- macOS 12(Monterey) 이상;"로그인 시 자동 실행"은 macOS 13 이상 필요
- Apple Silicon 또는 Intel 칩(해당하는 버전을 다운로드하세요)
- 인터넷 연결 필요(음성 인식은 클라우드에서 처리됩니다)

## 🗒 업데이트 내역

### 2.6.1 (2026-07-20)
- 중요 수정: "~를 찾아줘", "~를 비교해줘" 같은 명령형 문장을 음성 입력하면 AI 다듬기가 내용을 자신에 대한 지시로 오해해 대답해 버리는 문제(예: "죄송합니다, 도와드릴 수 없습니다" 출력)를 수정. 이제 항상 충실하게 다듬으며, 출력이 AI 답변처럼 보이면 자동으로 원문으로 되돌리는 안전장치도 추가——ChatGPT/Claude 등 AI 도구에 음성 입력하는 사용자는 꼭 업데이트하세요

### 2.6.0 (2026-07-20)
- **"자가 진화" 기능 추가**: 사용 기록을 자동 분석해 반복적으로 잘못 인식되는 단어를 찾아 교정 규칙을 제안합니다. 한 번 승인하면 이후 자동으로 수정——쓸수록 정확해집니다
- **TAIDE 로컬 다듬기 옵션 추가**: 대만 국가과학기술위원회의 TAIDE 모델 지원(Ollama를 통해 Mac에서 로컬 실행, 다듬기 텍스트가 컴퓨터 밖으로 나가지 않음). 설치 안내 포함. 음성 인식과 번역은 계속 Groq 사용
- 로컬 모델 리소스 최적화: 컨텍스트 길이 제한과 3분 유휴 시 자동 메모리 해제로 발열·메모리 낭비 방지
- 수정: 말한 내용이 숫자만일 때(예: "123") 오류가 발생하던 문제
- 수정: 숫자 위주의 중국어 문장이 외국어로 오판되어 영어로 번역되던 문제

### 2.5.1 (2026-07-19)
- 수정: API 키 등 대화 상자 입력란에서 Cmd+V 붙여넣기가 되지 않던 문제(마우스 오른쪽 클릭만 가능했음). Cmd+C/X/A/Z도 함께 수정

### 2.5.0 (2026-07-19)
- "대만어(실험적)" 인식 모드 추가 — 대만어로 말하면 출력 언어 설정에 따라 중/영/일/한 텍스트로 출력
- 음성 인식·다듬기/번역 모델 선택 기능 추가. 각 선택지에 개발사와 국가를 명시(기본 모델은 변경 없음)
- "요약 강도" 3단계 추가: 끄기(군말 제거만)/보수적(명백한 반복 병합)/적극적(핵심만 남기기)
- 인터페이스 언어에 日本語·한국어 추가(라이선스 약관 전문 포함). 시스템 따르기 또는 수동 지정 가능
- 메뉴 재구성: API 키·모델·요약·기록을 "고급 설정"으로 모아 메인 메뉴를 간결하게
- 녹음 음량이 너무 작을 때 조용히 건너뛰지 않고 마이크 입력 음량 확인을 안내

### 2.4.0 (2026-07-18)
- 첫 공개 릴리스: Apple 서명+공증 DMG, Apple Silicon 및 Intel 지원(macOS 12+)
